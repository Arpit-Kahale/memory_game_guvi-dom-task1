# Memory Game 

## About the Project

This is a simple **Memory Card Game** made using **HTML, CSS and JavaScript**.
The main idea of this project is to practice **DOM manipulation and basic game logic in JavaScript**.

In this game, the player flips cards and tries to find matching pairs. If two cards match, they stay open. If they do not match, they flip back after a short delay.

The game also includes a Restart button which allows the player to reset the game board and shuffle the cards again to start a new game.

---

## Technologies Used

* **HTML** – for creating the structure of the game
* **CSS** – for styling the cards and game layout
* **JavaScript (DOM)** – for handling card flip, matching logic and shuffling

---

## Features

* Cards flip when the player clicks on them
* Cards are shuffled randomly when the game starts
* Matching cards stay open
* Non-matching cards flip back after a short delay
* Restart button to start the game again
* Simple responsive layout that works on desktop and mobile

---

## How the Game Works

1. The game board contains several cards placed face down.
2. When a player clicks a card, it flips and shows its value.
3. The player can flip two cards at a time.
4. If both cards have the same value, they stay open.
5. If they are different, they flip back after a short delay.
6. The goal is to find all matching pairs.

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open **index.html** in your browser.
4. The game will start and you can begin playing.
